# Get Party Of Player

Returns the party of the input player as List[Player]. If the player is not in a party, it will return a list containing only that player.



| Input Name | Input Type |
|-----------|-----------|
| Target | player |

| Output Name | Output Type |
|-----------|-----------|
| Party | List[player] |
