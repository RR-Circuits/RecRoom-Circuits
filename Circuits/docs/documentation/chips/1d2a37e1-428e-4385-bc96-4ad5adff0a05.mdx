# Quaternion Split

Split a quaternion into its four float components: x, y, z, and w. Congratulations, you have reached peak math.



| Input Name | Input Type |
|-----------|-----------|
| Quaternion | quaternion |

| Output Name | Output Type |
|-----------|-----------|
| X | float |
| Y | float |
| Z | float |
| W | float |
