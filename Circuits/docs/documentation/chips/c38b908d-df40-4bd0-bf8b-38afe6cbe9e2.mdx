# RGB To Color

Return a color from the given red, green, and blue inputs. All inputs are in the range [0, 1].



![Example banner](./assets/c38b908d-df40-4bd0-bf8b-38afe6cbe9e2.svg)

| Input Name | Input Type |
|-----------|-----------|
| Red | float |
| Green | float |
| Blue | float |

| Output Name | Output Type |
|-----------|-----------|
| Color | color |
